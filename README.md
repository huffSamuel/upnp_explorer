## Installation

Clone this repo with
```shell
$ git clone https://github.com/huffSamuel/upnp_explorer.git
```

then build the APK with
```shell
$ flutter build apk
```

and install it on your device with
```shell
$ adb install ./build/app/outputs/flutter-apk/app-release.apk
```
